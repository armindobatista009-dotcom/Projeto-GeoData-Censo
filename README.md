# Projeto GeoData Censo

Repositório para o projeto de visualização e consulta de dados do Censo.

Conteúdo:
- Código do servidor: `server.js`
- Consultas de dados: `data-query.js`
- Arquivos de dados grandes em `data/` (ignorados por padrão no .gitignore)

Como rodar:

```bash
npm install
node server.js
```

Observações:
- Ajuste `db-config.js` conforme necessário.
- Opcional: use `start.bat` no Windows para iniciar o servidor.
